\section{引言}
% ═══════════════════════════════════════════════════════════

新闻推荐系统面临的核心挑战之一，是用户兴趣随时间的动态漂移——阅读偏好在事件、上下文和时间的共同作用下，呈现非线性的连续变化。传统推荐系统以点击率最大化为目标，容易导致信息茧房：用户反复接收同类内容，长期体验反而下降。Lin 等人~\cite{lin2025interest} 在序列推荐中显式建模了兴趣漂移；Wang 等人~\cite{wang2025intent} 从意图层面论证了超越物品级相似度的多样化推荐的有效性；Chen 等人~\cite{chen2018neural} 的 Neural ODE 为连续时间动态建模提供了数学基础。然而，如何将连续时间兴趣演化与意图覆盖规划结合，仍是一个开放问题。

本文提出 Intent-Drift News Planner，一个四模块流水线框架：Qwen3-Embedding~\cite{qwen3embedding} 语义编码、GRU 双塔排序器、Neural ODE 世界模型和 Beam Search 意图覆盖规划器。我们在 MIND 数据集~\cite{wu2020mind} 上进行了系统评估。

需要说明的是，MIND 数据集已转为私有访问。本文使用的数据来自 PaddleRec 仓库的公开镜像，并非 Microsoft MIND 官方下载。我们在此镜像基础上构建了训练集和验证集。

% ═══════════════════════════════════════════════════════════
\section{数据集}
% ═══════════════════════════════════════════════════════════

MIND~\cite{wu2020mind} 是新闻推荐领域广泛使用的基准数据集，由微软于 2019 年发布。本文使用 PaddleRec 仓库提供的 MINDsmall 子集，包含 96,074 篇新闻。每条新闻含标题、摘要和类别（如 sports、finance、lifestyle）。用户行为以 impression 为单位组织，包含浏览历史和当前曝光的候选新闻（含点击/未点击标签）。

\subsection{样本构建}

\textbf{Ranker 样本。}对每个 impression，保留全部被点击的正例。训练集按正负比 1:4 采样负例，验证集保留全部负例。每条样本包含：impression\_id、user\_id、history\_ids（最近 50 条）、candidate\_id、category 和 label。

\textbf{世界模型样本。}用户状态定义为最近 50 条浏览历史的加权平均（最近一篇权重 50，最早一篇权重 1）。多步预测（$N=3$）时，从同一 impression 取连续 $N$ 个点击作为动作序列，目标为加入全部 $N$ 个点击后的加权平均状态。仅当 impression 含 $\ge 3$ 个正例时才生成多步样本。

验证集采用 impression 级随机采样（种子 42），保证每个 impression 的候选组结构完整。

\subsection{数据局限性}

多步世界模型样本仅占总样本的约 3\%，原因是 MIND 的正样本率较低（约 3.5\%），大多数 impression 的点击数不足以构成 3 步序列。这一稀疏性是数据集本身的特性，我们在实验部分进行了诚实呈现。

\section{方法}
% ═══════════════════════════════════════════════════════════

系统采用四阶段流水线（图~\ref{fig:arch}）。给定浏览历史，Qwen3 将新闻编码为语义向量，加权平均得到用户状态；GRU 排序器筛选候选；Neural ODE 预测每篇候选对状态的长期影响；Beam Search 结合意图覆盖输出 Top-5。

\subsection{语义编码}

采用 Qwen3-Embedding-0.6B~\cite{qwen3embedding} 作为冻结编码器，将标题与摘要拼接后取均值池化，输出 256 维向量。所有新闻向量经 L2 归一化后以 NPZ 格式存储，运行时按 ID 查表。

\subsection{GRU 双塔排序器}

用户塔使用 GRU（隐藏维度 256）对最近 50 条新闻嵌入序列编码，取最后时间步输出为用户向量。新闻塔直接使用候选嵌入。双塔余弦相似度经线性层和 sigmoid 映射为点击概率。训练使用 BCE 损失，AdamW（$lr=10^{-3}$），batch size 256，10 epochs。

\subsection{Neural ODE 世界模型}

世界模型预测推荐行为对用户状态的影响，使用 Neural ODE 建模：
\begin{equation}
\frac{d\mathbf{u}}{dt} = f_\theta(\mathbf{u}, \mathbf{a})
\end{equation}
其中 $f_\theta$ 为两个 512 维隐藏层的 MLP（ReLU 激活）。状态积分采用四阶 Runge-Kutta（RK4），多步时依次对每步动作执行积分。

\textbf{Drift-Aware 训练。}传统 MSE 损失对状态预测过于宽容——加权平均表征高度平滑，模型仅需近似恒等映射即可获得极低 MSE。我们提出将损失分解为状态级和漂移级两部分：
\begin{equation}
\mathcal{L} = \mathcal{L}_{\text{state}} + \mathcal{L}_{\text{drift}} + 0.5\,\mathcal{L}_{\text{cos}} + \mathcal{L}_{\text{cos-drift}}
\end{equation}
其中 $\mathcal{L}_{\text{drift}}$ 为预测漂移 $\Delta\hat{\mathbf{u}}$ 与真实漂移 $\Delta\mathbf{u}$ 的 MSE，$\mathcal{L}_{\text{cos-drift}}$ 为漂移方向余弦损失。漂移监督迫使模型学习状态变化的方向和幅度，而非退化为恒等映射。训练 10 epochs，batch size 256，AdamW（$lr=10^{-3}$）。

\subsection{Beam Search 规划器}

规划器以 Beam Search（horizon=3，beam\_width=5，branching\_factor=8）展开搜索。初始一条路径（用户当前状态），每轮对活跃路径展开最高分候选，调用世界模型预测未来状态，计算综合奖励（相关性、漂移增益、意图覆盖、长期对齐、类别多样性），保留最优 beam 进入下一轮。最终 Coverage Reranker 对最优路径贪心去重，输出 Top-5。

\section{实验}
% ═══════════════════════════════════════════════════════════

\subsection{实验设置}

实验环境：NVIDIA RTX 5090（32 GB），PyTorch 2.7，CUDA 12.8。Ranker 使用全量 950 万样本训练（10 epochs，最终 loss=0.437）。世界模型使用 31 万条多步样本训练（Drift-Aware loss，最终复合 loss=0.023）。评估在验证集上随机采样 2,000 个 impression（种子 42，约 78K ranker 样本、47K WM 样本）。评估指标包括 Accuracy、AUC、Recall@5、NDCG@5、Intent Coverage@5（IC@5）、Intra-List Similarity@5（ILS@5）、HitRate@5（HR@5），以及世界模型的 State/Drift Cosine 和 MSE。

\subsection{世界模型漂移预测}

我们在 46,792 条多步样本上进行三方对比（表~\ref{tab:wm}）。恒等映射的 State Cosine 为 0.9857，是系统下限。纯 MSE 训练的 Neural ODE 将 State Cosine 推至 0.9968——但这实为退化：模型学习近似恒等映射。证据是其 Drift Cosine 仅 0.8712。

Drift-Aware 训练牺牲了部分 State 精度（Cosine 降至 0.9927），换来了漂移方向预测的显著提升：Step 3 的 Drift Cosine 从 0.8712 跃升至 0.9789，且每一步均高于纯 MSE 基线。Drift Cosine 的递进趋势（0.57 $\to$ 0.80 $\to$ 0.98）验证了多步 RK4 积分的信号累积能力。

\subsection{结果小结}

Ranker 提供了可靠的候选基础（AUC 0.701）；世界模型在 Drift-Aware 训练下学到了漂移方向（Drift Cosine 0.57 $\to$ 0.98）；Planner 是覆盖率提升的核心（IC@5 +29.9 pp）。纯 MSE 的漂移余弦仅 0.87 验证了显式漂移监督的必要性。

\section{讨论}
% ═══════════════════════════════════════════════════════════

\subsection{训练目标对世界模型行为的影响}

纯 MSE 训练的漂移余弦仅 0.87 而 Drift-Aware 达到 0.98——这一差距揭示了训练目标设计对模型行为的决定性影响。纯 MSE 优化下，模型的最优策略是近似恒等映射（State Cosine 高达 0.9968），因为状态本身变化极小。Drift-Aware 的漂移监督打破了这一退化，迫使模型关注状态演化的方向性信息。

\subsection{与现有工作的关联}

Lin 等人~\cite{lin2025interest} 在离散时间步上建模兴趣漂移——本文将其推广为连续时间 Neural ODE，避免了固定时间粒度的假设。Wang 等人~\cite{wang2025intent} 提出意图多样化框架——本文的 Intent Coverage 和 Coverage Reranker 是其直接实现。Chen 等人~\cite{chen2018neural} 的 Neural ODE 提供了连续动态建模基础。

\subsection{未来工作方向}

本文的发现为若干改进提供了明确动机。第一，加权平均表征的平滑性提示可探索更具区分度的状态编码：将历史窗口从 50 降至 10--20 可使新点击权重从 11\% 提升至 25--35\%；用可学习 Transformer 聚合器替代固定加权平均；或使用最近状态替代全局平均以提高灵敏度。这些改变需同步调整 Ranker。第二，当前多步世界模型仅使用同一 impression 内的点击——MIND 的跨曝光时序数据可构建更长的兴趣演化链，同时缓解数据稀疏性。第三，Beam Search 推理开销可通过模型蒸馏或剪枝优化。第四，在更大数据集上的验证也是自然方向。

% ═══════════════════════════════════════════════════════════
\section{结论}
% ═══════════════════════════════════════════════════════════

本文提出 Intent-Drift News Planner，在 MIND 数据集（PaddleRec 镜像）上进行了系统评估。GRU 排序器达到 94.48\% 准确率和 0.701 AUC。Drift-Aware Neural ODE 在漂移余弦上从 0.57 递增至 0.98，验证了多步信号累积的价值；纯 MSE 训练的漂移余弦仅 0.87 进一步证实了显式漂移监督的必要性。Beam Search 规划器将 Top-5 意图覆盖率从 66.5\% 提升至 96.4\%。未来工作将探索更灵敏的状态编码和跨曝光时序建模。
